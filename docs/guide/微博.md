# 获取 User ID

## 电脑端

打开微博个人主页，链接栏的数字即为 ```User ID```

<a data-fancybox title="chrome_weibo" href="/Snipaste_2021-07-25_17-20-06.png">![chrome_weibo](/Snipaste_2021-07-25_17-20-06.png)</a>

## 手机端

在微博官方客户端（第三方同理）打开个人首页，右上角三个点，复制链接即可得到 ```User ID```
