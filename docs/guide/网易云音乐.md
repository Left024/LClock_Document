# 获取 ID

## 电脑端

打开个人主页，```home?id=``` 之后的即为 ```ID```

<a data-fancybox title="" href="/Snipaste_2021-07-27_11-04-24.png">![](/Snipaste_2021-07-27_11-04-24.png)</a>

## 手机端

打开官方客户端个人主页，右上角三个点-分享按钮，点击复制链接即可获得 ```ID```
