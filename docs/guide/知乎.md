# 获取 ID

## 电脑端

打开知乎个人主页，链接栏 ```people``` 之后即为 ```ID```

<a data-fancybox title="" href="/Snipaste_2021-07-26_18-32-15.png">![](/Snipaste_2021-07-26_18-32-15.png)</a>

## 手机端

官方客户端打开个人主页，点击右上角分享按钮，复制链接即可获取 ```ID```
